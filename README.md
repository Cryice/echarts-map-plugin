##   制作过程

***乡镇级行政区划不支持***

    需要在此特殊说明，目前部分城市和省直辖县是无法返回街道的，这块我们在进行优化，若您需要还请耐心等待一下。
    例如：广东-东莞、海南-文昌市、广东-中山

### 采用高德地图制作
 http://gallery.echartsjs.com/editor.html?c=xr1ohZxAax

在控制台打印出日志文件，保存为xx.json

### 采用高德Web服务API

http://lbs.amap.com/api/webservice/guide/api/district/
